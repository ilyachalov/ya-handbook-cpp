# ya-handbook-cpp

Я изучаю бесплатный [учебник](https://education.yandex.ru/handbook/cpp) (хендбук) по основам языка программирования C++ от [«Яндекс Образование»](https://education.yandex.ru/about). В этом репозитории буду размещать примеры из глав хендбука в том виде, в котором я их набираю и компилирую, пока читаю главу. Также я буду размещать здесь решения задач, которые есть после многих (но не всех) глав хендбука.

Проект начат в 2025 году. Всего в хендбуке 5 частей, из которых первая и пятая — служебные и поэтому короткие. Части состоят из глав, которые я еще называю «уроками», хотя освоение многих из этих глав очень часто не влезает в пределы одного академического часа. Примеры и решения задач раскладываю по папкам, соответствующим главам-урокам.
